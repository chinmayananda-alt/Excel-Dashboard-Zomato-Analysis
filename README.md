Zomato Data Analysis 🍽️📊

Project Overview

This project focuses on exploratory data analysis and visualization of Zomato restaurant data to understand restaurant distribution, cuisine popularity, customer preferences, and rating patterns across locations.

The analysis is performed using Microsoft Excel with pivot tables, pivot charts, and interactive visuals to transform raw restaurant data into meaningful business insights.
